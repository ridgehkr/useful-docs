# `useCSSCustomProp()`

This hook allows you to monitor the current value of a CSS custom property in your React components. CSS custom properties, also known as CSS variables, provide a way to define reusable values in your CSS styles. This hook tracks the value of a given custom property in real-time, enabling you to reflect its current value within your React component.

## Usage

1. Make sure you have `@ridgehkr/useful` installed in your application. If not, you can get started by following the [installation instructions](/#getting-started).

2. Import and use the `useCSSCustomProp` hook in your component:

```tsx
import { useCSSCustomProp } from '@ridgehkr/usefulHooks'
```

### Example

`useCSSCustomProp` will track the value of a CSS custom prop for you and returns a string reflecting the current value:

```css /--theme-color/ filename="CSS"
:root {
  --theme-color: #ffcc00;
}
```

In your React component:

```tsx /--theme-color/ filename="MyComponent.js"
import { useCSSCustomProp } from '@ridgehkr/useful'

function MyComponent() {
  const themeColor = useCSSCustomProp('--theme-color')

  // "Current theme color: #ffcc00"
  return <div>Current theme color: {themeColor}</div>
}
```

## API Reference

### Parameters

The `useCustomCSSProp` hook accepts two parameters:

| Property          | Type                                                                          | Required                    | Description                                                                        |
| :---------------- | :---------------------------------------------------------------------------- | :-------------------------- | :--------------------------------------------------------------------------------- |
| **`property`**    | `string`                                                                      | yes                         | The name of the CSS custom property you want to track.                             |
| **`rootElement`** | [`HTMLElement`](https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement) | no (default: document root) | A reference to the root element where the CSS custom property should be monitored. |

### Return Value

The hook returns the current value of the specified CSS custom property as a string. If the custom property does not exist or is not set, it returns an empty string.

## How it Works

This hook retrieves the current value of the CSS custom property and sets up a [`MutationObserver`](https://developer.mozilla.org/en-US/docs/Web/API/MutationObserver) to efficiently monitor changes to the custom property. When a change is detected, the hook updates its state with the new value, causing your component to re-render with the updated value.

## Tips

- Ensure that the CSS custom property you want to track is defined in your CSS styles, either inline or in an external stylesheet.
- If you need to track multiple CSS custom properties in the same component, you can use multiple instances of the `useCustomCSSProp` hook with different property names.
- Custom property names are case-sensitive, so ensure your specified property matches the exact name used in your CSS.
