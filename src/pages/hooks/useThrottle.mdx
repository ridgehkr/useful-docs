# useThrottle

The `useThrottle` hook is a custom React hook that allows you to limit the frequency of a value changing to once every specified interval (defaults to 400ms). This can be especially useful when dealing with fast-changing values, such as those updated in response to user input or external events. The hook helps prevent excessive re-renders and improves performance by ensuring that the value updates are throttled to a specified interval.

## Installation

1. Make sure you have `@ridgehkr/useful` installed in your application. If not, you can get started by following the [installation instructions](/#getting-started).

2. Import and use the `useThrottle` hook in your component.

## Usage

The `useThrottle` hook takes two parameters: the `value` that you want to throttle and an optional `interval` in milliseconds. It returns the throttled value, which is updated no more frequently than the specified `interval`.

This is a basic example where `useThrottle` is used to limit the update frequency of a search input's value:

```tsx filename="SearchForm.tsx"
import React, { useState } from 'react'
import { useThrottle } from '@ridgehkr/useful'

export const SearchInput = () => {
  const [inputValue, setInputValue] = useState<string>('')
  const throttledValue = useThrottle<string>(inputValue, 600)

  const handleInputChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    setInputValue(e.target.value)
  }

  return (
    <div className={style.demo}>
      <header className={style.header}>
        <h1>useThrottle</h1>
      </header>

      <p>
        <input onChange={handleInputChange} />
      </p>
      <p>
        Throttled input value: <strong>{throttledValue}</strong>
      </p>
    </div>
  )
}
```

The `inputValue` state value is updated whenever the user types in the input field. However, the actual rendering of the throttled value (`throttledValue`) will only occur at most once every 600 milliseconds. This prevents excessive re-renders when the user is rapidly typing.

## API Reference

### Parameters

| Property       | Type   | Description                                                         |
| :------------- | :----- | :------------------------------------------------------------------ |
| **`value`**    | any    | The value you want to throttle                                      |
| **`interval`** | number | The interval in milliseconds between updates of the throttled value |

### Return Value

| Property             | Type | Description                                                                 |
| :------------------- | :--- | :-------------------------------------------------------------------------- |
| **`throttledValue`** | any  | The throttled value, which is updated no more frequently than the interval. |
