# useStack

This hook manages a Stack abstract data type. A stack is a Last-In-First-Out (LIFO) data model where elements are added to and removed from the top only. You can learn more about how stacks work [on Wikipedia](<https://en.wikipedia.org/wiki/Stack_(abstract_data_type)>) or the [NIST](https://xlinux.nist.gov/dads/HTML/stack.html).

## Installation

To use this custom hook in your React project, you can follow these steps:

1. Make sure you have `@ridgehkr/useful` installed in your application. If not, you can get started by following the [installation instructions](/#getting-started).

2. Import and use the `useStack` hook in your component.

## Usage

```tsx /useStack/ filename="Stack.tsx"
import React from 'react'
import { useStack } from '@ridgehkr/useful'

export const Stack = () => {
  const stack = useStack<number>()

  useEffect(() => {
    console.log('useEffect', stack.items)

    stack.push(1)
    stack.push(2)

    console.group('Stack')
    console.log('Stack:', stack.items) // [1, 2]
    console.log('Peek:', stack.peek()) // 2
    console.log('Pop:', stack.pop()) // 2
    console.log('Size:', stack.size()) // 1
    console.groupEnd()
  }, [])

  return <p>See console for stack details.</p>
}
```

## API Reference

### Parameters

`useElementSize` does not accept any parameters.

### Return Value

`useStack` returns an object with the following methods:

| Property    | Signature         | Returns                                              | Description                                     |
| :---------- | :---------------- | :--------------------------------------------------- | :---------------------------------------------- |
| **`push`**  | `(item: T) => {}` | `void`                                               | Pushes an item onto the top of the stack        |
| **`pop`**   | `() => {}`        | The popped item or `undefined` if the stack is empty | Removes and returns the top item from the stack |
| **`peek`**  | `() => {}`        | The top item or `undefined` if the stack is empty    | Returns the top item from the stack             |
| **`size`**  | `() => {}`        | The number of items in the stack                     | Returns the number of items in the stack        |
| **`items`** | `() => {}`        | An array containing the items in the stack           | Returns the items in the stack                  |
