# `useElementSize()`

The useElementSize hook provides an easy way to track and retrieve the size of a DOM element. It monitors the changes in the size of a specified DOM element and updates the size information accordingly.

## Usage

1. Make sure you have `@ridgehkr/usefulHooks` installed in your application. If not, you can get started by following the [installation instructions](/#getting-started).

2. Import and use the `useElementSize` hook in your component:

```tsx
import { useElementSize } from '@ridgehkr/usefulHooks'
```

## Example

The `useElementSize` hook takes one parameter, a React `ref` for the element that you want to track the size of.

Here's an example of how you can use the `useElementSize` hook in your React component:

```tsx
import './App.css'
import { useElementSize } from '@ridgehkr/usefulHooks'

function App() {
  const { width, height, offsetWidth, offsetHeight } = useElementSize()

  return (
    <div>
      <h1>Random Password Generator</h1>

      <h2>{password}</h2>

      <p>
        <label>
          Length:
          <input
            type='number'
            min={1}
            value={length}
            onChange={(e) => setLength(parseInt(e.target.value))}
          />
        </label>
      </p>

      <p>
        <label>
          <input
            type='checkbox'
            checked={uppercase}
            onChange={(e) => includeUppercase(!!e.target.checked)}
          />{' '}
          Include Uppercase Characters? {uppercase ? 'Yes' : 'No'}
        </label>
      </p>

      <p>
        <label>
          <input
            type='checkbox'
            checked={symbols}
            onChange={(e) => includeSymbols(!!e.target.checked)}
          />{' '}
          Include Symbols? {symbols ? 'Yes' : 'No'}
        </label>
      </p>

      <p>
        <label>
          <input
            type='checkbox'
            checked={numbers}
            onChange={(e) => includeNumbers(!!e.target.checked)}
          />{' '}
          Include Numbers? {numbers ? 'Yes' : 'No'}
        </label>
      </p>
    </div>
  )
}

export default App
```

In this example, the `usePasswordGenerator` hook is used to generate a random password based on the current settings. The password is updated automatically whenever the settings are changed.

## Parameters

`usePasswordGenerator()` accepts an object of arguments to initially define the properties of the generated password. All of these parameters can be changed later using the [functions returned](#return-value) by the hook.

| Parameter       |  Type   | Required? | Description                                                           |
| :-------------- | :-----: | --------: | --------------------------------------------------------------------- |
| **`len`**       | number  |     false | The length of the password to generate                                |
| **`symbols`**   | boolean |     false | Whether or not to include symbols in the generated password           |
| **`numbers`**   | boolean |     false | Whether or not to include numbers in the generated password           |
| **`uppercase`** | boolean |     false | Whether or not to include uppercase letters in the generated password |

### Example

```tsx
// Initialize the hook with a password with a length of 24 and including symbols, numbers, and uppercase letters
const { password } = usePasswordGenerator({
  length: 24,
  symbols: true,
  numbers: true,
  uppercase: true,
})
```

## Return Value

The `usePasswordGenerator` hook returns an object with the following properties:

| Property                                 |   Type   |                                             Description |
| :--------------------------------------- | :------: | ------------------------------------------------------: |
| **`password`**                           |  string  |    The generated password based on the current settings |
| **`length`**                             |  number  |                    The length of the generated password |
| **`setLength(length: number)`**          | function |                   A function to set the password length |
| **`symbols`**                            | boolean  |                     Whether or not symbols are included |
| **`includeSymbols(include: boolean)`**   | function |           A function to toggle the inclusion of symbols |
| **`numbers`**                            | boolean  |                     Whether or not numbers are included |
| **`includeNumbers(include: boolean)`**   | function |           A function to toggle the inclusion of numbers |
| **`uppercase`**                          | boolean  |           Whether or not uppercase letters are included |
| **`includeUppercase(include: boolean)`** | function | A function to toggle the inclusion of uppercase letters |

### Return Function Examples

The returned functions provide a means to change the length and character types of the generated password. For example:

```tsx
// Set the password length to 24
setLength(24)

// Include symbols in the generated password
includeSymbols(true)

// Include numbers in the generated password
includeNumbers(true)

// Include uppercase letters in the generated password
includeUppercase(true)
```
